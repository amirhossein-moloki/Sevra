# نیازمندی‌های صفحات فرانت‌اند پروژه Sevra

بر اساس بررسی دقیق بک‌اند (مدل‌های Prisma و API Endpointها)، صفحات و قابلیت‌های زیر برای پیاده‌سازی سمت فرانت‌اند (در سه بخش پنل مدیریت، سایت عمومی و پنل مشتری) استخراج شده است:

## ۱. پنل مدیریت (Admin Panel) - برای مدیران و پذیرش

### الف. احراز هویت (Auth)
- **صفحه ورود:** ورود با شماره موبایل و رمز عبور یا کد یکبار مصرف (OTP).
- **صفحه فراموشی رمز عبور.**

### ب. داشبورد (Dashboard)
- **خلاصه وضعیت:** نمایش تعداد رزروهای امروز، درآمد کل، و رزروهای جدید.
- **نمودارها:** نمودار درآمد ۳۰ روز گذشته و مقایسه عملکرد کارکنان.

### ج. تقویم رزرو (Booking Calendar)
- **نمای تقویم:** نمایش رزروها به صورت روزانه، هفتگی و ماهانه.
- **مدیریت رزرو:** امکان ثبت رزرو جدید (In-Person)، ویرایش زمان، تغییر وضعیت (Confirm, Complete, No-Show) و لغو رزرو.
- **جریان پرداخت:** ثبت دستی مبالغ دریافتی یا مشاهده وضعیت پرداخت‌های آنلاین.

### د. مدیریت کارکنان (Staff Management)
- **لیست کارکنان:** مدیریت نقش‌ها (Manager, Receptionist, Staff).
- **مدیریت شیفت‌ها:** تنظیم ساعت کاری برای هر روز هفته برای هر آرایشگر.
- **پروفایل عمومی:** تنظیم بیوگرافی، آواتار و نمایش یا عدم نمایش در سایت عمومی.

### ه. مدیریت خدمات (Service Management)
- **لیست خدمات:** تعریف نام خدمت، مدت زمان، قیمت و فعال/غیرفعال بودن.
- **دسته‌بندی:** (در صورت نیاز) گروه‌بندی خدمات.

### و. مدیریت مشتریان (CRM)
- **بانک اطلاعات مشتریان:** لیست تمام مشتریانی که رزرو داشته‌اند.
- **تاریخچه مشتری:** مشاهده رزروهای قبلی، نظرات ثبت شده و یادداشت‌های اختصاصی برای هر مشتری.

### ز. مدیریت محتوا و سایت (CMS)
- **صفحات:** مدیریت محتوای صفحات اصلی، درباره ما، تماس با ما و صفحات سفارشی.
- **گالری تصاویر:** آپلود و مدیریت تصاویر نمونه کار (Before/After)، لوگو و کاور سالن.
- **تنظیمات سئو:** مدیریت عنوان و توضیحات متا برای هر صفحه.
- **لینک‌های اجتماعی:** تنظیم لینک اینستاگرام، تلگرام و واتس‌اپ.

### ح. تنظیمات سالن (Salon Settings)
- **اطلاعات پایه:** نام، اسلاگ (Slug)، آدرس فیزیکی و مختصات روی نقشه.
- **قوانین رزرو:** فعال‌سازی رزرو آنلاین، تایید خودکار رزروها، و تنظیمات منطقه زمانی.

### ط. تحلیل و گزارشات (Analytics)
- **گزارش ریونیو:** تحلیل درآمد به تفکیک خدمت و بازه زمانی.
- **عملکرد کارکنان:** تعداد رزروهای تکمیل شده و امتیاز دریافتی هر آرایشگر.

### ی. پورسانت‌ها (Commissions)
- **لیست پورسانت‌ها:** مشاهده لیست کارمزدهای پلتفرم برای هر رزرو آنلاین.
- **پرداخت پورسانت:** ثبت فیش یا پرداخت آنلاین بدهی سالن به پلتفرم.

---

## ۲. سایت عمومی (Public Website) - برای مشتریان

### الف. صفحه اصلی سالن (Salon Landing Page)
- نمایش بنر اصلی، معرفی سالن، و خدمات برتر.
- نمایش آدرس، ساعات کاری و لینک‌های شبکه‌های اجتماعی.

### ب. قیف رزرو آنلاین (Booking Funnel)
1. **انتخاب خدمت:** لیست خدمات به همراه قیمت و زمان.
2. **انتخاب آرایشگر:** نمایش کارکنان فعال.
3. **انتخاب زمان:** نمایش اسلات‌های خالی بر اساس الگوریتم Availability.
4. **تایید و پرداخت:** دریافت مشخصات مشتری (نام و موبایل) و اتصال به درگاه پرداخت (Zarinpal/Stripe).

### ج. صفحات محتوایی
- نمایش صفحات ساخته شده در CMS (مانند گالری تصاویر یا درباره ما).

---

## ۳. پنل مشتری (Customer Dashboard - اختیاری/آینده)
- **تاریخچه رزروها:** مشاهده وضعیت رزروهای فعلی و قبلی.
- **ثبت نظر (Reviews):** امکان ثبت امتیاز و نظر برای رزروهای تکمیل شده.
- **مدیریت پروفایل:** ویرایش نام و اطلاعات تماس.

---

## موارد فنی پیشنهادی برای فرانت‌اند:
- **Internationalization (i18n):** پشتیبانی از تقویم شمسی و میلادی.
- **Responsive Design:** اولویت بالا برای نسخه موبایل (مشتریان عمدتا با موبایل رزرو می‌کنند).
- **Real-time Notifications:** استفاده از WebSocket یا Polling برای آپدیت لحظه‌ای تقویم مدیریت هنگام ثبت رزرو آنلاین.
